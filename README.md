# vscode-inline-types

A Visual Studio Code extension that will inline-render inferred types in TypeScript documents.

![Inline rendering in action](assets/demo.png)

## Install

Inside VSCode, press `Ctrl+P`, and enter:

```
ext install inline-types
```
